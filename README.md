This is an old version of HotSwap that I decompiled the .jar file with the source code, as it was not available. 

Do note: this is not a maintained version, and I have no plans to update it. It is only here for reference, and to be used as a base for anyone who wants to make their own version of HotSwap.

The main reason for releasing this version publically (other than because I can), is on the off chance someone is attempting to figure out ray tracing etc on an older version of Minecraft.

This mod was originally built for Forge 1.16.2. I have attempted to add comments on what I believe certain functions do, and if I can think of flawed logic, I will do my best to point it out,
treat the 1.16.2 branch as essentially a self audit.

# HotSwap
HotSwap is a mod that allows you to swap to the best weapon in your hotbar while mining.
